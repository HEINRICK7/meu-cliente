# UX References - Meu Cliente

Este arquivo registra apenas a inspiracao visual usada para orientar a direcao de produto.

## Referência visual adotada

Usamos como inspiracao um app mobile de agenda com:

- cabecalho coral/laranja
- fundo claro levemente cinza
- cards grandes e arredondados
- card de destaque amarelo
- cards brancos para agendamentos
- botoes circulares escuros
- tipografia limpa
- bastante espacamento
- visual moderno, simples e amigavel
- aparencia de app premium, mas facil de usar

## Regra de uso

- A referencia serve apenas como inspiracao visual.
- Nao copiar a interface exatamente.
- Extrair apenas os padroes de clima, hierarquia e sensacao visual.
- A direcao oficial do produto continua sendo `docs/UX_UI.md`.
